<h1>Бэкенд</h1>

## Установка

```bash
$ npm install
```
Создать файлы .env.production и .env.development на основе .env.example

## Запуск

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Тестирование

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
